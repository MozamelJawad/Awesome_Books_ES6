# 📖 Awesome_Books_ES6 <a name="about-project"></a>

**Awesome_Books_ES6** The *Awesome Books ES6* project is developed in HTML, CSS, and JavaScript ES6, and it is a learning project in Microverse that can help users to store the collections of books in local storage.
